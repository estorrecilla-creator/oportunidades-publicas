## 🎯 PASO 1: DOMINIO

### Opción 1: Namecheap (LO QUE RECOMENDÉ)
```
Costo: €15-20/año
Ventajas: Interface clara, soporte bueno, renovación fácil
Desventajas: Publicidad agresiva, upsells
```

### ✅ Opción 2: IONOS (1&1) - MÁS BARATO
```
Costo: €10-15/año (primer año), luego €15-20
Link: https://www.ionos.es/dominios
Ventajas: 
  ✅ Más barato
  ✅ Interface clara
  ✅ Soporte decente
  ✅ Incluye WhoisGuard gratis (privacidad)
Desventajas: Servicio al cliente a veces lento
```

### Opción 3: GRATIS (NO RECOMENDADO)
```
Freenom: .tk, .ml, .ga (GRATIS pero:)
  ❌ No profesional
  ❌ Google lo penaliza
  ❌ No vale para SaaS
```

**RECOMENDACIÓN:** IONOS (€15 máximo, muy buen balance)

```
Costo: GRATIS

Ventajas:
  ✅ Gratis y potente

### ✅ Opción 2: Firebase (Google) - ALTERNATIVA GRATIS
```
Costo: GRATIS

Desventajas: Poco espacio

## 💳 PASO 5: PASARELA DE PAGOS

### Opción 1: CREEM (LO QUE RECOMENDÉ)
```
Costo: Comisión 2-3% por transacción
Setup: €0
Ventajas: Enfocada en España
Desventajas: Puede ser más cara a largo plazo
```

### ✅ Opción 2: STRIPE - MÁS BARATO
```
Costo: 2.9% + €0.30 por transacción (MEJOR QUE CREEM)
Setup: €0
Link: https://stripe.com/es
Ventajas:
  ✅ MÁS BARATA (mismo % o menos)
  ✅ Mejor para suscripciones
  ✅ Webhook robusto
  ✅ Test mode gratis
  ✅ Dashboard excelente
Desventajas: Requiere verificación (5-7 días)
```

### Opción 3: PAYPAL
```
Costo: 3.49% + €0.49
Ventajas: Conocida
Desventajas: Más cara
```

### Opción 4: REDSYS (La Caixa, Banco Santander)
```
Costo: 0% setup + comisión según banco
Ventajas: Bancos españoles
Desventajas: Más complicado de setup
```

**RECOMENDACIÓN:** STRIPE (más barata + mejor documentación)
